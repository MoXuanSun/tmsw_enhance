# 東方月灵传增强脚本

## 一、 这是什么
这是一张星际争霸游戏大厅自定义地图的增强脚本

主要目的是优化原图、修复bug

为了方便，本项目使用纯galaxy编写

## 二、 Q&A
### Q1：是否对原地图的数据有修改？
没有
### Q2：可以提交pr和issue吗？
可以
### Q3: 本拓展的意义？
原图没有一个完整训练营模式，测试起来很麻烦，本拓展会添加一个类似于dota2的训练营模式，供测试使用

原图的优化很差，我的电脑在游玩时经常达到80-90度，而玩其他的图很少达到这个温度，且很容易受到卡顿影响

原图的bug很多，比如双击bug和天赋不能再次选择等等，本拓展会尝试修复这些bug
